# ANALYSE_D_UN_PARC_AUTOMOBILE
le project de l'analyse d'un parc automobile est réalisé dans le cadre de la filière Ingénierie des Données, au sein du module Traitement des données sur Python,  L’objectif est de mettre en pratique les outils étudiés afin d’analyser un fichier contenant des informations sur plusieurs voitures.
